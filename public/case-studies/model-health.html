The Problem

Model quality assurance on large BIM projects is one of the most time-consuming and error-prone tasks in the coordination cycle. Senior coordinators manually open models, visually inspect naming conventions, check parameter completeness, verify workset assignments, and cross-reference against the BIM Execution Plan — a process that typically takes 1–2 full days per discipline per audit cycle.

Core Challenges

  • Manual audits take 1–2 days per discipline — 5+ days for a full multi-discipline check
  • No quantifiable metrics — "the model is messy" is not actionable feedback
  • Issues discovered late in the coordination cycle, after they've compounded
  • Different auditors apply different standards — inconsistent quality gates

Project Context

Built for the Al Noor Tower Phase 2 project, where the BIM coordination team needed a way to quantify model quality, track improvement over time, and provide discipline leads with specific, actionable issue lists rather than vague "clean up your model" requests.

The Solution

A browser-based model health analytics platform that ingests Revit model audit data, scores health across 8 standardized QA categories, and presents results through interactive radar charts, severity-coded issue tables, and exportable JSON reports.

Health Score

Overall 0–100 score with letter grades (A/B/C), calculated as a weighted average across all QA categories.

Radar Analysis

Spider chart showing strengths and weaknesses across 8 categories — instantly identify which areas need attention.

Issue Tracker

Filterable issues table with Critical/Warning/Info severity levels, element IDs, and discipline tagging.

8 QA Categories

Naming, Parameters, Placement, Rooms, Coordinates, Views, Families, and Detailing — the full BEP compliance stack.

Discipline Breakdown

Bar charts showing issue distribution by discipline — data-driven accountability for each trade.

Export Reports

One-click JSON export with full audit data, category scores, and issue details for project records.

Engineering Highlights

  • Health score uses weighted category averages — critical categories (Families, Parameters) carry 1.5x weight
  • Radar chart provides instant visual "shape" of model quality — flat areas = weak spots
  • Issue severity classification follows ISO 19650 non-conformance grading principles
  • Export format designed to be machine-readable for CI/CD model validation pipelines
