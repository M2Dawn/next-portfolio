# BIM Automation Tool

Replaced a 6–8 hour manual export cycle with a single-click Revit API add-in. Batch-exports 200+ views to DWG with intelligent naming conventions, extracts schedule data to structured CSV, and eliminates rework caused by inconsistent file management.

## The Challenge

Coordination teams were spending an entire day manually exporting sheets and schedules. This process was error-prone and tedious.

## The Solution

I built a custom Revit API add-in using C# and WPF to automate this entirely.

- **Batch Processing**: Extracts 500+ views with a single click.
- **Naming Conventions**: Automatically applies project-specific naming rules.
- **Data Extraction**: Pulls schedule data directly into CSV.

## Results

- **65% Time Reduction**: From 6-8 hours down to under 2 hours.
- **Error Free**: Eliminated human errors in naming and sheet selection.
